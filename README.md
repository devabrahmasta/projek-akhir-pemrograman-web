PETUNJUK RUNNING WEBSITE
1. Buka file connection.php pada direktori config/connection.php
2. Ubah variabel hostname, username, password, port, dan database sesuai konfigurasi pada XAMPP dan nama database di MySQL
3. Jalankan Module Apache dan MySQL pada XAMPP
4. Jalankan direktori yang sesuai di browser. contoh: http://localhost/praktikum/project_akhir/views/main_page/index.php
